# AI Social Media Post Generator

An AI-powered web app that generates creative and platform-tailored social media posts using **Google Gemini API**.  
Built with **React + TypeScript + Vite**, it allows users to input raw text, select target platforms, and instantly generate optimized post content.

## Setup Instructions

### 1.Clone the repo

git clone https://github.com/gangamahesh246/AI-Social-Media-Post-Generator-.git
cd AI-Social-Media-Post-Generator-

### 2.Install dependencies

npm install

### 3.Create a .env file

VITE_GEMINI_API_KEY=your_api_key_here

### 4.Run the project

npm run dev
